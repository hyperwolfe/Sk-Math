# Sk-Cuts
Skript addon for new shortcut syntax with optimizations to run as good or sometimes even better than skript!

Syntax for V1.0: 

%number% += %number%  
%number% -= %number%  
%number% *= %number%  
%number% /= %number%  
%number% ^= %number%  

Please don't redistribute as your own (illegal)!
